# PilotKids

**Kelajak muhandislari shu yerda boshlanadi**

Zamonaviy robototexnika va muhandislik ta'lim platformasi — React.js + Vite + Tailwind CSS bilan qurilgan.

## Texnologiyalar

- React.js 19
- Vite 8
- Tailwind CSS 4
- React Router DOM
- Framer Motion
- AOS (Animate On Scroll)
- Recharts
- Lucide React Icons

## O'rnatish

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Xususiyatlar

- Futuristik robotics laboratory dizayni
- Dark/Light mode
- Glassmorphism UI
- Framer Motion animatsiyalar
- JWT-ready autentifikatsiya strukturasi
- Responsive dizayn (mobile → ultra-wide)
- O'zbek tilida kontent
